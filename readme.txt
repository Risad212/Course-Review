Adds a course review and rating system for LearnPress courses.

== Description ==

Course Review is a LearnPress addon that allows students to submit reviews and ratings for courses.

Features:
- Course rating system (1–5 stars)
- Student reviews and comments
- LearnPress course integration
- Lightweight and extendable structure
- Developer-friendly architecture

This plugin is designed as an extension for LearnPress LMS.

== Installation ==

1. Upload the plugin folder to /wp-content/plugins/
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure LearnPress is installed and activated
4. Start extending the review system

== Frequently Asked Questions ==

= Does this work without LearnPress? =
No, LearnPress is required for this plugin to function.

= Can I customize the review system? =
Yes, the plugin is built with a modular structure for easy customization.
